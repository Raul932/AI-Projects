# Machine Learning Coursework

A collection of 5 ML/DS/DL assignments built during my AI studies at UBB.  
Covers data cleaning, EDA, regression, clustering, TensorFlow, CNNs and more.

## Assignments Overview
1. Data Cleaning, Feature Engineering & Movie Recommendation System
2. Regression Models & Hyperparameter Tuning
3. Clustering (K-Means, DBSCAN) & Logistic Regression
4. TensorFlow Neural Networks
5. Custom CNN & AlexNet implementation

## Tech Stack
Python, Pandas, NumPy, Matplotlib, scikit-learn, TensorFlow, Keras

## Structure
Each assignment is in a separate folder and contains a Jupyter Notebook.
